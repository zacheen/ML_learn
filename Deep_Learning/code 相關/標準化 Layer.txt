標準化 layer

BatchNorm2d 標準化的一種方式
	對每個 Channel 計算自己的 均值以及方差
	新的值 = (舊的值 - 均值)/方差  (這只是概念 非正確的公式 正確的公式反正也看不懂)
	
	作用 :
		就是標準化結果
		
	理論 :
		就算每個 Pixel 的值 1.同時+1 或 2.同時*1.5 結果應該要相同
			可是我記得每個 pixel / 255 結果會比較好
				這有可能是 float 計算比較精準的結果
