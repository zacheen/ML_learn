公司電腦建立 pytorch CPU 環境
    建立環境
        conda create --name pytorch_cpu python=3.7
        conda activate pytorch_cpu
        conda install pytorch torchvision cpuonly -c pytorch (一樣沒有安裝 torchaudio)
        pip install pillow Cython matplotlib pandas opencv-python
        
        ipynb 必備
            conda install -n pytorch_cpu ipykernel --update-deps --force-reinstall
    
    如果已經 activate 環境了 要再進入
        可以從 anaconda navigator 進入
    
    關閉
        conda deactivate
